# Authentication

The **Authentication** documentation outlines the methods for securing and managing access to Claude Code, an AI development platform integrated with Anthropic’s models. It details three primary authentication pathways: **Claude for Teams/Enterprise** (ideal for centralized billing, SSO, and role-based permissions), **Claude Console** (for API-based billing with user role assignments), and **Cloud Provider Authentication** (via Amazon Bedrock, Google Vertex AI, or Microsoft Foundry). Each method involves distinct setup steps—such as subscribing, inviting users, or configuring SSO—and emphasizes security best practices like credential management and system requirements. The documentation also highlights differences between **Claude for Teams** (self-service, collaboration-focused) and **Claude for Enterprise** (enterprise-grade security, compliance, and policy controls).
