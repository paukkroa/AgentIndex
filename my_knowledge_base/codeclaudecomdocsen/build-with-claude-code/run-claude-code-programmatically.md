# Run Claude Code programmatically

The **"Run Claude Code programmatically"** documentation provides a comprehensive guide to executing Claude Code’s agent capabilities via the **Agent SDK**, accessible through a **CLI tool** (formerly "headless mode") or **Python/TypeScript packages** for full programmatic control. The CLI (`claude -p`) enables non-interactive execution by appending the `-p` flag to any `claude` command, supporting features like **structured output** (`--output-format json` or `stream-json`), **auto-approved tools** (`--allowedTools`), **conversation continuation** (`--continue`), and **custom system prompts**. For advanced use cases, the **Agent SDK** (Python/TypeScript) offers deeper integration, including **structured outputs, tool approval callbacks, and native message objects**. Key examples include extracting structured data (e.g., JSON schemas), streaming responses, and automating workflows via CLI flags or SDK methods. The page also covers **session metadata**, **structured output validation**, and **conversation persistence** for seamless programmatic interactions.
