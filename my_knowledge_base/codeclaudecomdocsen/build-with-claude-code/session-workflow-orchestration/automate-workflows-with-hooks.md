# Automate workflows with hooks

Hooks in Claude Code are **user-defined shell commands** that execute at predefined lifecycle stages (e.g., input prompts, code edits, or configuration changes) to enforce deterministic automation. They enable **project-specific rules**, **task automation**, and **third-party integrations** by running scripts at critical events like user input requests, file edits, or context compaction. Hooks support **shell commands**, **structured JSON output**, and **conditional logic** via **matchers** (e.g., file patterns, agent types). Advanced use cases include **prompt-based hooks** (LLM-driven decisions) and **agent-based hooks** (multi-agent workflows). Troubleshooting covers common issues like misfiring hooks, JSON validation errors, or infinite loops, with debugging tools and reference docs for event schemas, async hooks, and **Model Context Protocol (MCP)** integrations.
