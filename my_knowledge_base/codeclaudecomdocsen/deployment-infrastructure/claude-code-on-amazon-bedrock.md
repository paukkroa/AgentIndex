# Claude Code on Amazon Bedrock

Claude Code on Amazon Bedrock integrates Anthropic’s AI models with AWS’s Bedrock platform, enabling developers to deploy, configure, and manage Claude AI agents within the AWS ecosystem. This documentation outlines a **step-by-step deployment workflow**, starting with **prerequisites** (AWS account access, Bedrock-enabled models, AWS CLI, and IAM permissions) and proceeding through **use case submission** (mandatory for first-time users via the Bedrock console). **AWS credential configuration** is supported via multiple methods (CLI, environment variables, SSO, or Bedrock API keys), with advanced options for credential refresh and corporate identity providers. Key configuration steps include **Claude Code setup** (model version pinning to avoid breaking changes), **IAM role assignment** for secure access, and **AWS Guardrails** for compliance. Troubleshooting and additional resources are provided for debugging deployment issues, ensuring seamless integration with Bedrock’s infrastructure.
