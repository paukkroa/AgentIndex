# Enterprise deployment overview

This page provides a comprehensive **Enterprise Deployment Overview** for Claude Code, guiding organizations through selecting the optimal deployment strategy—whether via **Anthropic’s direct offerings (Claude for Teams/Enterprise)** or **cloud provider integrations (Amazon Bedrock, Google Vertex AI, Microsoft Foundry)**. It highlights key distinctions: **Claude for Teams** (self-service, collaboration-focused, $150/seat) and **Claude for Enterprise** (SSO, role-based permissions, compliance APIs, and managed policies) cater to varying organizational needs. For infrastructure-specific requirements, the page compares deployment options across **billing models (PAYG vs. fixed pricing), supported regions, authentication methods (SSO/API keys), and cost tracking tools**. Additional sections cover **proxy/gateway configuration, security policies, model version pinning, and best practices** like investing in documentation, guided usage, and leveraging **Model Configuration Protocol (MCP)** for integrations. The page concludes with actionable **next steps** for organizations to streamline deployment and ensure scalability.
