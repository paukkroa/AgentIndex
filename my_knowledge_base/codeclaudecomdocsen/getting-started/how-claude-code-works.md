# How Claude Code works

Claude Code is an **agentic AI assistant** embedded in your terminal, designed to automate workflows beyond coding—such as documentation, builds, file searches, and research—by leveraging an **autonomous agentic loop** comprising three iterative phases: **context gathering** (e.g., analyzing codebases), **action execution** (e.g., editing files or running tests), and **result verification** (e.g., validating changes). The system dynamically adapts to tasks, chaining actions and refining outputs based on real-time feedback. Key components include **large language models** for reasoning and **tools** for execution, with **context management** via skills/subagents, checkpoints for safety, and permissions to control access. Users can **interrupt, steer, or delegate** tasks, while Claude maintains a **conversational, iterative workflow**—exploring before implementing, verifying against provided references, and avoiding rigid dictation. Extensibility features (e.g., MCP hooks) enable customization, while **session persistence** (resume/fork) and **context window limits** ensure scalability. Best practices emphasize specificity, verification, and gradual delegation to optimize collaboration.
