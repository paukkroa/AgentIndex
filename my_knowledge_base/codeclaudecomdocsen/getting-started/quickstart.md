# Quickstart

This **Quickstart Guide** provides a step-by-step introduction to **Claude Code**, an AI-powered coding assistant integrated across multiple platforms (CLI, web, VS Code, JetBrains, Slack, GitHub Actions, and GitLab CI/CD). It begins with **prerequisites** (terminal, project, and Claude subscription) before guiding users through **eight core steps**:
1) **Installation** via CLI (native, Homebrew, or WinGet) with auto-updates for native installs;
2) **Account login** via Claude Console or cloud providers;
3) **First session initiation** to explore AI capabilities;
4) **Asking questions** and receiving code suggestions;
5) **Making edits** with AI-assisted changes;
6) **Git integration** for version control;
7) **Bug fixes/feature additions** with AI guidance;
8) **Exploring workflows** (e.g., debugging, refactoring). Additional sections cover **essential CLI commands**, beginner tips, and next steps, ensuring users quickly leverage Claude Code’s AI-driven development tools.
