# Claude Code GitHub Actions

Claude Code GitHub Actions integrates AI-driven automation into GitHub workflows by enabling **AI-assisted code review, PR generation, feature implementation, and bug fixes** via `@claude` mentions in PRs/issues. Built on the **Claude Agent SDK**, it leverages Anthropic’s Claude models (defaulting to **Sonnet**, with optional **Opus 4.6**) to analyze code, enforce project standards, and execute tasks programmatically. Key features include **customizable automation via prompts**, **multi-cloud support** (AWS Bedrock, Google Vertex AI, or direct Claude API), and **advanced configuration** for CLI arguments, security, and performance optimization. The setup supports **quickstart, manual deployment, and upgrades from beta**, with troubleshooting guides for common issues like authentication errors or CI misconfigurations. Use cases span **code generation, testing, documentation, and CI/CD optimization**, with best practices for cost management and workflow efficiency.
