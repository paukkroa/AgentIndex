# Claude Code GitLab CI/CD

Claude Code GitLab CI/CD integrates **Anthropic’s Claude AI** into GitLab’s CI/CD pipelines, enabling **automated code generation, issue-to-MR conversion, and bug fixes** via GitLab CI jobs. Built on the **Claude Code CLI and Agent SDK**, this beta feature leverages GitLab’s native runners to execute AI-driven workflows while adhering to branch protection and approval workflows. Key capabilities include **project-aware code suggestions** (aligned with `CLAUDE.md` guidelines), **multi-cloud model support** (Claude API, AWS Bedrock, or Google Vertex AI), and **enterprise-grade security** (OIDC/Workload Identity Federation for cloud integrations). Setup requires minimal configuration: adding a CI job to `.gitlab-ci.yml` and a masked variable, with options for **quickstart (API-only)** or **manual setup (production-ready)**. Advanced features include **customizable behavior via CLI parameters**, **cost optimization**, and **troubleshooting guides** for common issues (e.g., authentication errors, MR permissions).
