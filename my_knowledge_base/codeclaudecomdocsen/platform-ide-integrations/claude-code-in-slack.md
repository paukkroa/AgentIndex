# Claude Code in Slack

Claude Code in Slack integrates AI-powered coding assistance directly into Slack workspaces, enabling seamless delegation of development tasks via natural language commands. Built atop the existing **Claude for Slack** app, this integration automatically routes coding-related messages (triggered by `@Claude`) to **Claude Code on the web**, eliminating context-switching. Key features include **automatic intent detection**, **context-aware workflows** (leveraging Slack threads, channels, and GitHub repos), and **session-based collaboration** with real-time notifications. Access control is granular, supporting **user-level, workspace-level, and channel-based permissions**, while **repository selection** occurs dynamically via GitHub integration. The UI includes **App Home dashboards**, **message actions** (e.g., code execution, file edits), and **session management tools**. Troubleshooting covers common issues like authentication errors, session failures, or repository misalignment. Best practices emphasize **clear request phrasing**, **Slack vs. web use-case differentiation**, and **collaborative debugging** to maximize productivity. Limitations (e.g., session timeouts, beta features) are documented for transparency.
