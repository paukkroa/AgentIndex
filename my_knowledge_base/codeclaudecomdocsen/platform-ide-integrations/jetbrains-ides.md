# JetBrains IDEs

The **JetBrains IDEs** section of Claude Code documentation outlines the integration of Claude Code with JetBrains’ suite of IDEs (IntelliJ IDEA, PyCharm, Android Studio, WebStorm, PhpStorm, and GoLand) via a dedicated plugin. Key features include **interactive diff viewing** (displaying changes within the IDE’s diff tool instead of the terminal), **automatic selection context sharing** (sending the current file/selection to Claude), **file reference shortcuts** (`Cmd+Option+K`/`Alt+Ctrl+K` for inserting `@File#L1-99` syntax), and **diagnostic sharing** (automatically transmitting IDE errors/linting issues to Claude). Installation is straightforward via the **JetBrains Marketplace**, requiring a plugin install and IDE restart. Usage spans **in-IDE terminals** (via `claude` command) or **external terminals** (using `/ide` flag), with optional configurations for **remote development (WSL)**, **keybindings (ESC key)**, and **troubleshooting** (e.g., plugin detection, command errors). Security considerations and compliance resources are also referenced.
