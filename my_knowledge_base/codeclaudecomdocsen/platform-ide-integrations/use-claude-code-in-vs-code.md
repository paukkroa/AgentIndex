# Use Claude Code in VS Code

The **Claude Code VS Code extension** integrates Anthropic’s AI-powered coding assistant directly into the Visual Studio Code IDE, offering a seamless workflow for developers. This native extension enables real-time code generation, editing, and debugging with Claude’s AI, including interactive plan review, auto-accept edits, and file-specific line-range mentions via `@-mentions`. Key features include **multi-tab conversation management**, **conversation history persistence**, **Git integration** (commits, pull requests, and parallel worktrees), **plugin customization**, and **terminal output integration**. Users can also **switch between extension and CLI modes**, configure settings via VS Code’s UI, and leverage **checkpoints** for undo/redo functionality. The extension supports **third-party AI providers** (e.g., AWS Bedrock, Google Vertex AI) and includes troubleshooting guides for common issues like installation failures or API connectivity. Ideal for developers seeking AI-assisted productivity within VS Code, it bridges the gap between human intent and automated code execution while maintaining security and privacy controls.
