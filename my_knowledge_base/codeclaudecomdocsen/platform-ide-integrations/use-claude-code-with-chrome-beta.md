# Use Claude Code with Chrome (beta)

Claude Code’s **Chrome extension (beta)** integrates browser automation into the Claude Code CLI or VS Code, enabling seamless workflows between coding and browser tasks. Users can execute browser actions (e.g., form filling, data extraction, or UI testing) directly from the CLI, with real-time execution in a visible Chrome/Edge window. Key features include **live debugging** (DOM/console inspection), **authenticated app interactions** (Google Docs, Gmail, etc.), **multi-site workflows**, and **session recording** via GIFs. The extension leverages existing browser sessions (e.g., logged-in states) and pauses for manual intervention at CAPTCHAs/logins. Supported browsers: **Chrome/Edge only** (beta); WSL and other Chromium browsers unsupported. Prerequisites: Chrome/Edge installed, with prerequisites outlined in the [Getting Started](https://docs.en/overview) section.
