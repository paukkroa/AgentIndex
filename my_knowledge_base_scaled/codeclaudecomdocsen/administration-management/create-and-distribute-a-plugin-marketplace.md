# Create and distribute a plugin marketplace

This guide provides a comprehensive walkthrough for creating and distributing a **custom plugin marketplace** within the Claude Code ecosystem. It covers the entire lifecycle: defining a `marketplace.json` schema with **required fields** (e.g., plugin metadata, sources) and **optional metadata** (e.g., descriptions, icons) to catalog plugins from **Git repositories, local paths, or relative URLs**. Key sections include **strict mode validation**, **hosting options** (GitHub recommended, with support for private repos), and **team-specific restrictions** (e.g., enforced marketplaces, version resolution via release channels). Advanced topics address **troubleshooting** (e.g., authentication failures, URL-based path issues) and **validation testing** to ensure seamless plugin discovery, updates, and installation. The guide also integrates with **analytics, security, and cost monitoring** for enterprise use cases.
