# Set up Claude Code

The **Set Up Claude Code** documentation provides a comprehensive guide for installing, configuring, and maintaining the Claude Code application across macOS, Windows, and Linux platforms. It outlines **system requirements** (OS versions, 4GB+ RAM, internet access, and supported countries) and **additional dependencies** (e.g., Node.js for legacy npm setups). Installation methods include **native CLI scripts** (auto-updating), **Homebrew** (manual upgrades required), and **WinGet** (Windows-specific). The guide covers **platform-specific setup** (e.g., Alpine Linux dependencies), **authentication**, and **version management** (manual updates, disabling auto-updates, or uninstalling). Optional sections address **binary integrity**, **configuration cleanup**, and tailored workflows for **individuals vs. teams**. Post-installation, users access Claude Code via terminal or GUI, with troubleshooting links for common issues.
