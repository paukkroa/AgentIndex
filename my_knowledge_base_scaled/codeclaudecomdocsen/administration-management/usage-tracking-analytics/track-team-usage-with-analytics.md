# Track team usage with analytics

Claude Code’s **Analytics Dashboard** provides granular insights into developer productivity and tool adoption across **Teams/Enterprise** and **API-based** plans. For **Teams/Enterprise**, admins access usage metrics (e.g., lines of code accepted, suggestion acceptance rates, daily active users) alongside **GitHub-integrated contribution metrics** (PRs, shipped code, and leaderboards) via [claude.ai/analytics/claude-code](https://claude.ai/analytics/claude-code). Features include **CSV data exports**, **contribution attribution** (with configurable tagging criteria, time windows, and excluded files), and **ROI measurement** (e.g., power user identification, adoption tracking). **API customers** (via [platform.claude.com/claude-code](https://platform.claude.com/claude-code)) focus on **spend tracking** and **team insights**. Setup requires **Owner role access** and GitHub app installation for contribution metrics (beta), while usage data is universally available. Advanced features like **PR attribution notes** and **programmatic data access** enhance custom reporting.
