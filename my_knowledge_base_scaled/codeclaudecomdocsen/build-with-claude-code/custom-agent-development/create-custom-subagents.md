# Create custom subagents

The **Create Custom Subagents** documentation outlines how to develop specialized AI assistants within Claude Code to handle niche tasks autonomously. Subagents operate in isolated contexts with custom system prompts, restricted tool access, and granular permissions, enabling task delegation from the main conversation. Key features include configurable scopes (user/project), model selection (e.g., cost-efficient Haiku), tool restrictions, and advanced capabilities like persistent memory, hooks for conditional logic, and parallel execution. Topics cover setup (frontmatter fields, file structure), delegation triggers (`/agents` command), security (permission modes), and optimization patterns (chaining, context isolation). Example subagents (e.g., code reviewers, data scientists) demonstrate real-world applications, while troubleshooting and delegation rules ensure seamless integration. Ideal for modularizing workflows, enforcing constraints, and reducing costs via targeted AI delegation.
