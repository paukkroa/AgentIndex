This section outlines the methods for integrating and extending Claude Code’s functionality through third-party tools and plugins. It covers two primary approaches: **direct tool integration via MCP (Multi-Tool Protocol)**—enabling HTTP, SSE, local, or stdio-based connections to external services—and **plugin discovery/installation** through centralized marketplaces, which provide prebuilt, vetted extensions to enhance Claude Code’s capabilities. Together, these mechanisms empower developers and users to seamlessly expand Claude Code’s workflows, automate tasks, and access specialized APIs or services.
