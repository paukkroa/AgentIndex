# Discover and install prebuilt plugins through marketplaces

This page explains how to **discover and install prebuilt plugins** for Claude Code via plugin marketplaces, which act as centralized catalogs for third-party extensions. Marketplaces streamline plugin discovery and installation by separating the process into two steps: **adding the marketplace** (registering the catalog) and **installing individual plugins** (browsing and selecting extensions). The **official Anthropic marketplace** (`claude-plugins-official`) is preloaded and accessible via the `/plugin` command, offering plugins categorized into **code intelligence** (e.g., LSP tool integration for code navigation, type checking, and refactoring) and **external integrations** (e.g., GitHub, Git hosts, or local/remote plugins). Users can also **add custom marketplaces** (GitHub, local paths, or remote URLs) and manage plugins via an **interactive interface** or **CLI commands**. Features include **auto-updates**, **team marketplace configurations**, and troubleshooting for common issues like plugin command recognition or code intelligence errors. The page also links to resources for **creating and distributing your own marketplaces** and **extending Claude’s capabilities** with skills, hooks, and MCP servers.
