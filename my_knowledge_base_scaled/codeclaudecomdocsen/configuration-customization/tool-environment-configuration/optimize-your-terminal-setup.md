# Optimize your terminal setup

This page provides a comprehensive guide to optimizing your terminal setup for **Claude Code**, covering customization options for themes, line breaks, notifications, and advanced workflows. Key features include **theme synchronization** via `/config` and **custom status lines** for contextual info (e.g., model, directory). Line breaks can be managed via **Quick Escape (`\` + Enter)**, **Shift+Enter** (native in iTerm2/WezTerm), or **custom keybindings** (e.g., Option+Enter for VS Code). For **notifications**, iTerm2 users can enable system alerts via preferences, while **custom hooks** allow advanced logic integration. Large inputs are optimized via **file-based workflows** (avoiding direct pastes) and **Vim Mode** support (limited subset of Vim keybindings via `/vim` or `/config`). Terminal-specific tweaks (e.g., macOS Terminal’s Meta key) and VS Code limitations are also addressed.
