# Common workflows

This page outlines **practical, actionable workflows** for leveraging **Claude Code** in daily development tasks, covering **code exploration, debugging, refactoring, testing, documentation, and session management**. Key features include **Plan Mode** (for safe, structured code analysis and refactoring), **extended thinking mode** (deep reasoning with step-by-step breakdowns), and **session tools** (resuming conversations, naming sessions, and parallel worktrees via Git). Workflows also integrate with **CI/CD pipelines (GitHub Actions/GitLab)**, **IDE extensions (VS Code, JetBrains)**, and **Unix-style utilities** (piping input/output). Example prompts and configurations (e.g., default Plan Mode) are provided for immediate adoption. Advanced topics like **worktree cleanup** and **notification systems** ensure seamless workflow automation.
