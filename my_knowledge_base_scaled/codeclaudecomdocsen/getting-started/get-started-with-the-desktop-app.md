# Get started with the desktop app

The **Claude Code Desktop** page provides a structured guide to launching and utilizing the graphical interface for Claude Code, emphasizing its advanced features like **visual diff review, Git worktree isolation, file attachments, and remote task execution** (via cloud VMs or SSH). The app supports **three core tabs**:
- **Chat** (general AI conversations, akin to claude.ai),
- **Cowork** (autonomous background agent for cloud-based tasks),
- **Code** (interactive coding assistant with real-time local file access and change approvals).
Key steps include **installation** (platform-specific downloads for macOS/Windows, with ARM64 limitations), **sign-in** (via Anthropic account), and **initial setup** (activating the Code tab, requiring a Pro/Max/Teams/Enterprise subscription). The guide also addresses troubleshooting (e.g., 403 errors) and transitions from CLI users. Post-installation, users can **start a session** by selecting a local project folder or remote environment (SSH/cloud) to begin coding tasks.
