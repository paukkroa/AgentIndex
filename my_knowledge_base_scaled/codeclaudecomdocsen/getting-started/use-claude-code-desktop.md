# Use Claude Code Desktop

Claude Code Desktop provides a **graphical interface** for interacting with Claude Code’s AI-driven coding assistant, replacing the CLI with a **visual workflow** for code editing, review, and execution. Key features include **real-time diff viewing** with inline comments for change review, **customizable permission modes** (e.g., manual approval for edits), **parallel session management** with Git isolation, and **multi-surface continuity** (e.g., switching between desktop and web). It supports **local, SSH, and remote sessions**, integrates with **third-party tools** (GitHub, Slack, VS Code, JetBrains), and offers **enterprise-grade controls** like SSO, device policies, and managed settings. Advanced users can **extend functionality** via plugins, skills, or custom tools, while troubleshooting guides address common issues (e.g., authentication errors, Git/LFS conflicts). The desktop version mirrors CLI capabilities but adds **visual feedback** and **collaborative features**, though some CLI-only tools remain unavailable.
