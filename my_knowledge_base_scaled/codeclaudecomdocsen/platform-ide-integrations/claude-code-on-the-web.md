# Claude Code on the web

Claude Code on the web is an AI-powered development environment integrated with Anthropic’s Claude AI, enabling real-time code execution, debugging, and collaboration directly in a browser. This platform supports seamless task teleportation between web and terminal interfaces, allowing developers to run, inspect, and modify code via a **diff view** for change review. Key features include **cross-platform integration** (Chrome extension, VS Code, JetBrains IDEs, GitHub Actions, etc.), **environment customization** (package managers, databases, cloud platforms, and Linux distributions), and **security controls** (network policies, proxy settings, and access levels). Users can share sessions via Enterprise/Teams, Max, or Pro accounts, with configurable **cloud environments**, **persistent environment variables**, and **version control** support. Limitations and best practices (e.g., network access, dependency management) are documented to ensure secure, efficient workflows.
