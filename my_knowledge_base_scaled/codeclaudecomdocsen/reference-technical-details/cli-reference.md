# CLI reference

The **CLI reference** for Claude Code provides a comprehensive guide to its command-line interface (CLI), covering core commands, customizable flags, and advanced configurations for workflow automation. Key features include:
- **Core commands** like `claude` (interactive REPL), `claude -p` (SDK queries), and session management (`-c` for continuation, `-r` for resuming by ID).
- **Flags for customization**, including `--add-dir` (directory access), `--agents` (dynamic subagent JSON), and `--append-system-prompt` (custom system prompt text).
- **Advanced controls** like `--permission-mode` (fine-grained tool access), `--chrome` (browser automation), and `--debug` (detailed logging).
- **Security/beta features** such as `--betas` (experimental API headers) and `--dangerously-skip-permissions** (for bypassing permission checks).
- **Integration tools** like `--allowedTools` (pre-approved tools) and `--append-system-prompt-file` (external prompt files).
- **Version management** via `claude update` and **MCP server configuration** with `claude mcp`.
