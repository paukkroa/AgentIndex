# Interactive mode

Claude Code’s **Interactive Mode** provides a powerful, keyboard-driven interface for real-time code editing, task automation, and system interaction, blending terminal efficiency with AI-assisted workflows. Key features include **Vim-inspired navigation and editing** (e.g., NORMAL mode for text manipulation, text objects like words/lines, and modal switching), **background task management** (via `Ctrl+B` for bash commands or agents, with confirmation via double `Ctrl+F`), and **AI-driven prompts** (e.g., MCP prompts for structured task execution, prompt suggestions, and PR review status tracking). Advanced controls like **reverse command history search (`Ctrl+R`)**, **verbose output toggling (`Ctrl+O`)**, and **model switching (`Alt+P`)** enhance productivity. Platform-specific shortcuts (e.g., macOS’s `Alt+B` for word navigation) require terminal configuration (e.g., enabling Option as Meta in iTerm2/Terminal.app). The mode supports **multiline input**, **theming**, and **task lists**, with built-in commands (e.g., `/terminal-setup` for extended thinking) and hooks for customization.
