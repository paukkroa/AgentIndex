# Troubleshooting

The **Troubleshooting** documentation for Claude Code provides a structured guide to resolving common technical issues across installation, authentication, performance, and IDE integration. It covers **cross-platform challenges**, including **Windows/WSL-specific errors** (e.g., OS detection failures, `node`/`npm` conflicts, and Git-Bash dependencies), **Linux/Mac permission errors**, and **native installation alternatives**. Sections address **authentication pitfalls** (repeated prompts, misconfigured credentials) and **configuration management** (file locations, resets). Performance troubleshooting includes **CPU/memory optimization**, **command hangs**, and **WSL networking mode adjustments**. IDE-specific issues (JetBrains detection, escape key failures) and **Markdown formatting inconsistencies** are also detailed, with best practices for generating clean output. The guide concludes with **help resources** and **reporting channels** for unresolved problems, ensuring comprehensive coverage for developers encountering roadblocks in deployment or usage.
