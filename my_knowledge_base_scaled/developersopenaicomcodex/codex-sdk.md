# Codex SDK

The **Codex SDK** enables programmatic control of Codex via a **TypeScript library**, ideal for **CI/CD automation, custom agents, internal tool integration, or application embedding**. Built for **Node.js 18+**, it offers a **server-side API** with thread-based workflows, allowing persistent conversations via `startThread()`/`resumeThread()` and dynamic prompt execution via `run()`. Key features include **thread continuity** (resuming past sessions), **flexible prompt chaining**, and **deep integration** with OpenAI’s Codex engine. Installation via `npm` (`@openai/codex-sdk`) provides access to a **comprehensive, interactive API** for advanced use cases beyond CLI/Web/IDE interfaces.
