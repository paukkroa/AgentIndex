The **Configuration & Setup** section provides a structured guide to customizing and deploying the Codex local client, catering to users from beginners to advanced users. It begins with foundational knowledge in *Config Basics*, covering essential configuration principles, before progressing to *Local Environments*, where users learn to set up worktrees and automation scripts for local development. For those requiring deeper customization, *Advanced Configuration* explores specialized options, while *Configuration Reference* serves as a definitive resource for the syntax and requirements of `config.toml` and `requirements.toml`. A *Sample Configuration* offers a ready-to-use template for quick adaptation, and platform-specific guidance in *Windows* ensures compatibility whether using native CLI/IDE or WSL2. Together, these resources ensure users can tailor Codex to their workflows efficiently.
