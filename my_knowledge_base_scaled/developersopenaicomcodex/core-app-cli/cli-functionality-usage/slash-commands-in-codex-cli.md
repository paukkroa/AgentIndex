# Slash commands in Codex CLI

Slash commands in the **Codex CLI** provide a streamlined, keyboard-driven interface for managing core functionality without leaving the terminal. By typing `/` in the composer, users access a popup menu featuring built-in commands like `/model` (switch models), `/permissions` (adjust approval settings), `/agent` (manage agent threads), and `/experimental` (toggle optional features). The CLI includes specialized commands such as `/sandbox-add-read-dir` (Windows-only directory access), `/compact` (summarize transcripts to optimize token usage), and `/diff` (review Git changes, including untracked files). Other utilities include `/apps` (integrate connectors), `/init` (generate `AGENTS.md` scaffolds), and `/feedback` (submit diagnostics). These commands enable real-time session control, session persistence, and troubleshooting, enhancing productivity for developers and operators.
