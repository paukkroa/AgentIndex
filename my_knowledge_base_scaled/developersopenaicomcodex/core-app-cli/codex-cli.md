# Codex CLI

Codex CLI is OpenAI’s locally deployable Rust-based coding assistant designed for terminal-based workflows, enabling real-time code reading, editing, and execution within selected directories. Built for speed and efficiency, it integrates with ChatGPT Plus/Pro/Business/Edu/Enterprise plans and supports macOS/Linux (with experimental Windows/WSL compatibility). Key features include **interactive terminal UI (TUI)**, model switching (e.g., GPT-5.3-Codex), image input for design specs, local code reviews, multi-agent collaboration, web search, and cloud task automation via `exec` scripting. Advanced capabilities like **Model Context Protocol (MCP)** extend tool integration, while **approval modes** enhance safety for edits/commands. Open-source (GitHub) and optimized for developer productivity.
