# Codex app features

The **Codex app** is a specialized desktop application designed for parallel development workflows, integrating **multi-project management**, **Git worktree support**, and **automations** to streamline collaboration and task execution. Users can switch between projects (mirroring CLI sessions) and isolate changes via worktrees or cloud environments, ensuring sandboxed development. The app leverages **agent skills** (shared across CLI/IDE) for customizable tooling, while **built-in Git tools** (diff, commit, PRs) and an **integrated terminal** enable seamless version control. **Automations** automate repetitive tasks (e.g., error analysis, reporting), and **modes** (Local/Worktree/Cloud) cater to localized or remote execution. Ideal for developers needing structured, scalable workflows with Git integration and AI-driven productivity.
