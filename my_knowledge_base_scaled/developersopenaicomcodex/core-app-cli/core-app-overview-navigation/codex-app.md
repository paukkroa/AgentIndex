# Codex app

The **Codex app** is a specialized macOS (Apple Silicon) desktop application designed for parallel development on **Codex threads**, integrating **worktree support, Git operations, and automation** to streamline workflows. Built for **ChatGPT Plus/Pro/Business/Edu/Enterprise** users, it enables **local-first AI-assisted coding** via **local threads** (with optional cloud sync) and **project-based organization**. Key features include:
- **Multi-threaded collaboration** (parallel task handling)
- **Git integration** (worktrees, branching, and version control)
- **Automated workflows** (e.g., bug fixes, code generation)
- **Project-specific AI prompts** (e.g., game development, debugging)
- **Local-first mode** (offline access with optional cloud sync)
- **Cross-platform roadmap** (currently macOS-only; sign up for Windows/Linux updates)

Requires **OpenAI API key or ChatGPT account** for authentication. Ideal for developers seeking **AI-powered, Git-native productivity**.
