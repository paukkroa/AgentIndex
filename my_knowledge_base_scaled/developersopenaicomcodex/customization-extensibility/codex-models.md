# Codex Models

The **Codex Models** page presents a curated selection of AI-powered coding models optimized for developers, engineers, and researchers. The **recommended models** include **gpt-5.3-codex**, the most advanced agentic coding model with superior reasoning, professional knowledge, and coding performance, available via **Codex app, CLI, IDE extension, and Cloud** (API access forthcoming). The **gpt-5.3-codex-spark** is a research preview model for **ChatGPT Pro users**, prioritizing ultra-fast, real-time coding iteration. The **gpt-5.2-codex** (now succeeded by gpt-5.3-codex) remains a robust alternative for real-world engineering tasks, offering full **CLI/SDK, IDE integration, Cloud, and API access**. For broader use cases beyond coding, **gpt-5.2** (a general-purpose agentic model) is highlighted as an alternative, though its features are collapsible for brevity. Each model’s capabilities are visually rated for **capability, speed, and integration support**, guiding users toward the best fit for their workflow.
