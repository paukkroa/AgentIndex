# Quickstart

The **Quickstart** guide introduces **Codex**, an AI-powered coding assistant integrated into ChatGPT (Plus, Pro, Business, Edu, and Enterprise plans) and accessible via API credits. Users can experience **free limited access** in ChatGPT Free/Go or **double the Codex rate limits** with paid subscriptions. Setup options include:
- **macOS App** (recommended for Apple Silicon users): Download, sign in (via ChatGPT or OpenAI API key), select a project, and interact locally.
- **IDE Extension**: Embed Codex directly into your development environment.
- **CLI**: Use Codex in the terminal for command-line interactions.
- **Cloud**: Access via browser without installation.

Key features include **local project analysis**, AI-driven code generation (e.g., building a Snake game from scratch), and integration with existing tools. API key sign-ins may limit certain features like cloud threads.
