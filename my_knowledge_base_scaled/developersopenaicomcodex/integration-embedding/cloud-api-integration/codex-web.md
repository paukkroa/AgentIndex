# Codex web

Codex is OpenAI’s AI-powered coding assistant designed to streamline development by reading, editing, and executing code autonomously. It integrates with GitHub via cloud-based processing (parallelizable tasks) to accelerate debugging, refactoring, and understanding unfamiliar codebases. Setup requires connecting a GitHub account to access repositories and generate pull requests. Key features include **prompt optimization** (structured input for precise outputs), **predefined workflows** (task delegation, PR creation), **cloud environment customization** (repo selection, tool integration), **IDE extension delegation** (cloud task initiation from editors like VS Code), **GitHub integration** (direct task triggering via `@codex` mentions), and **internet access controls** (restricting cloud environment connectivity). Available on Plus/Pro/Business/Edu/Enterprise plans, with Enterprise requiring admin configuration.
