# Codex IDE extension features

The **Codex IDE extension** integrates OpenAI’s Codex AI directly into VS Code, Cursor, Windsurf, and other VS Code-compatible editors, leveraging the same backend agent and configuration as the Codex CLI. Key features include **context-aware prompting**—where file references (e.g., `@file.tsx`) enhance prompt specificity—**model switching** (via a dropdown selector) to adjust complexity, and **approval modes** (`Agent`, `Chat`, or `Agent (Full Access)`) to control file/network access permissions. Users can also **adjust reasoning effort** (low/medium/high) to balance response depth and token usage. For large-scale tasks, **cloud delegation** offloads processing to a remote environment, preserving conversation context for seamless local follow-ups. Built-in **web search** (via a cached index) enables research without live page exposure, while cloud tasks are trackable via the Codex cloud interface. Ideal for developers seeking AI-assisted coding, debugging, and research within their IDE.
