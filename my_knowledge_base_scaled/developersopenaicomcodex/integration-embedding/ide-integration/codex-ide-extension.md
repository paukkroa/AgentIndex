# Codex IDE extension

The **Codex IDE extension** integrates OpenAI’s Codex AI assistant into VS Code, Cursor, Windsurf, and JetBrains IDEs (Rider, IntelliJ, PyCharm, WebStorm) to enhance coding workflows. It enables real-time code reading, editing, debugging, and execution via a sidebar chat interface, leveraging ChatGPT Plus/Pro/Business/Edu/Enterprise plans. Key features include seamless API key/ChatGPT sign-in, customizable sidebar placement (left/right), keyboard shortcuts, and automatic updates. Supported on macOS/Linux (Windows via WSL), it offers slash commands and context-aware task delegation to Codex Cloud. JetBrains users can authenticate via ChatGPT, API keys, or JetBrains AI subscriptions. Ideal for accelerating development, bug resolution, and navigating unfamiliar codebases.
