# Explore

The **"Explore"** page serves as a **task-based discovery hub** for developers, offering three **practical, scoped challenges** to engage with a codebase. Each task emphasizes **minimalist execution** while targeting distinct goals: **(1) Snake Game**—a constrained implementation of a classic game loop (movement, collision, scoring) with strict UI/dependency constraints, ideal for practicing modular logic and testing; **(2) Bug Fixing**—a disciplined workflow for reproducing, isolating, and verifying fixes with evidence-backed changes, prioritizing **confidence and minimal scope**; **(3) Viral Feature**—a high-impact, low-effort feature proposal (e.g., shareable links, referrals) aligned with existing product flows, with measurable outcomes. All tasks encourage **repo inspection**, **reusable patterns**, and **clear deliverables** (code changes, tests, or documentation) while avoiding scope creep. The gallery-style format enables **quick iteration** and **focused learning** for developers seeking hands-on practice in game dev, debugging, or growth hacking.
