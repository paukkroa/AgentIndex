# Command-line interface libraries¶

This page documents Python’s **standard library modules for building command-line and terminal interfaces**, enabling developers to create robust, user-friendly CLI tools. Key modules include:
- **`argparse`** (modern, object-oriented CLI argument parsing with subcommands and help generation),
- **`optparse`** (legacy, functional-style option parsing),
- **`getpass`** (secure password input handling),
- **`fileinput`** (streamlined iteration over multiple input files),
- **`curses`** (low-level terminal control for character-based UIs, including text input via `curses.textpad` and ASCII utilities in `curses.ascii`),
- **`curses.panel`** (stack-based panel management for layered terminal displays),
- **`cmd`** (foundational support for custom command-line interpreters with history and tab completion). These tools cater to everything from simple argument parsing to full-fledged terminal applications, leveraging Python’s built-in capabilities without external dependencies.
