# Functional Programming Modules¶

The **Functional Programming Modules** chapter in the Python Standard Library introduces core modules (`itertools`, `functools`, and `operator`) that enable functional programming paradigms. **`itertools`** provides efficient iterator-based tools like `chain()`, `groupby()`, and `combinations()` for lazy evaluation and complex looping patterns, including recipes for common functional operations. **`functools`** offers higher-order functions such as `partial()` for partial function application, `lru_cache()` for memoization, and `reduce()` for cumulative operations, alongside utilities for decorators and callable manipulation. **`operator`** exposes Python’s built-in operators (e.g., `itemgetter`, `attrgetter`, `add`, `mul`) as pure functions, enabling cleaner functional compositions and mappings. Together, these modules facilitate immutability, pure functions, and declarative workflows, bridging procedural and functional styles in Python.
