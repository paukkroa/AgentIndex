# Data Types¶

This chapter of the Python Standard Library documentation outlines **specialized and built-in data types** for advanced data handling. It covers **core built-in types** like `dict`, `list`, `set`, `frozenset`, `tuple`, `str`, `bytes`, and `bytearray`, alongside **module-specific types** for structured data manipulation. Key modules include:
- **`datetime`**: Comprehensive date/time handling with `timedelta`, `date`, `datetime`, `time`, and timezone-aware objects, including formatting via `strftime()`/`strptime()`.
- **`zoneinfo`**: IANA timezone support with configurable data sources and `ZoneInfo` class for timezone-aware operations.
- **`calendar`**: Utility functions for calendar operations (e.g., leap years, holidays).
- **`collections`**: High-performance container types like `ChainMap`, `Counter`, `deque`, `defaultdict`, and `namedtuple` for specialized use cases (e.g., chaining mappings, counting elements, or immutable named tuples).

Each module provides detailed examples, technical nuances (e.g., timezone serialization), and recipes for practical implementation.
