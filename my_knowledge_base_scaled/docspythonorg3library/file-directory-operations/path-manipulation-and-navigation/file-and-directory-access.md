# File and Directory Access¶

This chapter covers Python’s core modules for **file and directory operations**, including **path manipulation, file system traversal, and I/O handling**. Key modules include:
- **`pathlib`** (object-oriented filesystem paths with methods for path resolution, file operations, and permissions).
- **`os.path`** (legacy pathname utilities for compatibility).
- **`stat`** (interpreting file metadata via `stat()`).
- **`filecmp`** (directory/file comparison tools).
- **`tempfile`** (secure temporary file/directory generation).
- **`glob`/`fnmatch`** (Unix-style pattern matching for filenames).
- **`linecache`** (random text line access).
- **`shutil`** (high-level operations like copying, archiving, and directory management).

Each module supports cross-platform compatibility, with detailed sections on exceptions, URI handling, and performance optimizations (e.g., `shutil`'s platform-dependent copy methods).
