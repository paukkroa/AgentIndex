# File Formats¶

The **File Formats** documentation in Python’s standard library covers specialized modules for parsing and generating non-markup, non-email file formats. Key modules include:
- **`csv`** for reading/writing Comma-Separated Values (CSV) files, supporting custom dialects, reader/writer objects, and formatting parameters.
- **`configparser`** for handling INI-style configuration files, with features like interpolation, fallback values, and legacy API support.
- **`tomllib`** for parsing TOML (Tom’s Obvious Minimal Language) files, including type conversion tables and examples.
- **`netrc`** for processing `.netrc` files (storing authentication credentials).
- **`plistlib`** for generating/parsing Apple’s `.plist` (Property List) files in binary/XML formats.

Each module provides detailed APIs, examples, and edge-case handling, catering to structured data processing tasks.
