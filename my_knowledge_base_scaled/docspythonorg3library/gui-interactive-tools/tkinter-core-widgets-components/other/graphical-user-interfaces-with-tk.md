# Graphical user interfaces with Tk¶

The **Graphical User Interfaces (GUIs) with Tk** page explains Python’s built-in `tkinter` module, a cross-platform GUI toolkit integrated with Tcl/Tk. It serves as an object-oriented wrapper for Tk widgets, enabling Python developers to create platform-independent applications without writing Tcl code. Key features include:
- **Core `tkinter` package**: Offers a lightweight, bundled GUI solution with basic widgets (e.g., buttons, labels) and a structured architecture.
- **`tkinter.ttk` extension**: Provides modernized, themed widgets (e.g., `TTkButton`, `TTkFrame`) for improved aesthetics.
- **Documentation & Resources**: While official docs are limited, external guides (tutorials, books) and the Tk/Tcl reference manual supplement knowledge. Notable sections cover architecture, threading, layout managers (`pack`, `grid`), event bindings, and dialogs (e.g., file/color pickers).
- **Limitations & Alternatives**: Tk’s outdated UI (though improved in Tk 8.5) and lack of advanced features may prompt exploration of other frameworks (e.g., PyQt, Kivy).
