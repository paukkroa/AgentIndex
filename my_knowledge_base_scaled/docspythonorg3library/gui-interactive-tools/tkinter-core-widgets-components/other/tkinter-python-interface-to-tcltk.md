# tkinter— Python interface to Tcl/Tk¶

The `tkinter` module is Python’s standard GUI toolkit interface to **Tcl/Tk**, enabling cross-platform development for Unix (including macOS) and Windows. It integrates Tcl/Tk 8.6 (threaded by default) and extends its functionality with Pythonic abstractions, diverging from a simple wrapper. Key features include support for both legacy and modern themed UI components (introduced in Tcl/Tk 8.5), dynamic theming, and a layered architecture with `_tkinter` handling low-level Tcl/Tk interactions. Documentation emphasizes Python-specific enhancements, while Tcl/Tk’s core commands remain accessible via references. Optional on some distributions, `tkinter` requires manual installation if missing, with resources like [TkDocs](https://tkdocs.com/) and [Tcl/Tk Home Page](https://www.tcl.tk) offering tutorials and API details.
