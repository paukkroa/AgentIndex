# tkinter.scrolledtext— Scrolled Text Widget¶

The `tkinter.scrolledtext` module provides the **`ScrolledText`** class, a pre-configured text widget with an integrated vertical scrollbar, simplifying GUI development by combining a `Text` widget and scrollbar into a single, cohesive component. The widget is encapsulated within a `Frame`, inheriting geometry management methods (`Grid`/`Pack`) for flexible layout integration. Key attributes include `frame` (the container holding text and scrollbar) and `vbar` (the scrollbar widget). This abstraction eliminates manual scrollbar configuration, offering a streamlined alternative to manually pairing `Text` and `Scrollbar` widgets.
