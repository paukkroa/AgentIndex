# math— Mathematical functions¶

The **`math`** module in Python provides a comprehensive suite of **real-number mathematical functions and constants** aligned with the C standard library, excluding complex number support (use the [`cmath`](cmath.html) module for complex operations). It is designed for **floating-point arithmetic, number theory, and floating-point manipulation**, with functions categorized into:
- **Number-theoretic operations** (e.g., `comb()`, `factorial()`, `gcd()`, `isqrt()`, `lcm()`, `perm()`) for combinatorial and integer-based calculations.
- **Floating-point arithmetic** (e.g., `ceil()`, `fabs()`, `floor()`, `fmod()`, `trunc()`) for rounding, absolute values, and modular operations.
- **Floating-point manipulation** (e.g., `copysign()`, `frexp()`, `isclose()`, `ldexp()`, `ulp()`) for precision checks, bit-level operations, and floating-point comparisons.
- **Exponential/logarithmic/power functions** (e.g., `cbrt()`, `exp()`, `log()`, `pow()`) for advanced mathematical computations.
