# Numeric and Mathematical Modules¶

The **Numeric and Mathematical Modules** chapter of the Python Standard Library provides essential tools for numeric computations, abstract base classes, and mathematical operations. The [`numbers`](numbers.html) module establishes a hierarchical structure of numeric types (e.g., `Complex`, `Real`, `Integral`) to standardize arithmetic operations across implementations. The [`math`](math.html) module offers floating-point functions (e.g., trigonometric, logarithmic, constants) for real numbers, while [`cmath`](cmath.html) extends these to complex numbers, including polar conversions and hyperbolic functions. For precise decimal arithmetic, the [`decimal`](decimal.html) module supports arbitrary-precision fixed-point and floating-point operations, configurable rounding modes, and thread-safe contexts. Additionally, [`fractions`](fractions.html) handles rational numbers, and [`random`](random.html) generates pseudo-random numbers with bookkeeping and byte-related utilities. These modules collectively enable robust numerical analysis, scientific computing, and financial calculations.
