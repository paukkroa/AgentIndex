# Internet Data Handling¶

The **Internet Data Handling** chapter of the Python Standard Library provides modules for processing and managing data formats widely used across the internet, including email, JSON, mailbox manipulation, MIME types, and base encoding. The [`email`](email.html) module offers comprehensive tools for parsing, generating, and handling MIME-based email messages, supporting features like internationalized headers, character set encoding, and policy-based message construction. Submodules include `email.parser` (for parsing messages), `email.generator` (for MIME document creation), and `email.utils` (for utilities like address parsing). The [`json`](json.html) module enables serialization/deserialization of JSON data with compliance checks for standards like character encodings and NaN handling, alongside a command-line tool for manual JSON processing. The [`mailbox`](mailbox.html) module facilitates manipulation of mailboxes in formats like Maildir, mbox, and MH, with specialized objects for each format.
