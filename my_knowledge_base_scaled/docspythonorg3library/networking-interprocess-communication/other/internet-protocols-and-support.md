# Internet Protocols and Support¶

This chapter documents Python’s built-in modules for internet protocols and related functionalities, all implemented natively in Python. Central to these modules is the **`socket`** module, which provides low-level networking interfaces across platforms. Key modules include:

- **`webbrowser`**: Offers a command-line interface and browser controller objects for programmatically launching web browsers and handling URLs.
- **`wsgiref`**: A comprehensive WSGI (Web Server Gateway Interface) utility suite, including:
  - **`wsgiref.util`**: Environment utilities for WSGI applications.
  - **`wsgiref.simple_server`**: A lightweight HTTP server implementation.
  - **`wsgiref.validate`**: Tools for checking WSGI conformance.
  - **`wsgiref.handlers`**: Base classes for servers/gateways.
  - **`wsgiref.types`**: Static type checking utilities for WSGI.
