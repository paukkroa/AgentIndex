# Networking and Interprocess Communication¶

This chapter of the Python Standard Library documentation outlines core modules for **networking and interprocess communication (IPC)**, enabling both local and distributed process interactions. It covers low-level and high-level abstractions, including **asynchronous I/O** (`asyncio`), **socket-based networking** (`socket`/`ssl` for TLS/SSL), **I/O multiplexing** (`select`/`selectors`), **signal handling** (`signal`), and **memory-mapped files** (`mmap`). Modules like `socket` and `ssl` facilitate cross-machine communication via networking protocols, while `signal` and `mmap` are restricted to local processes. The chapter bridges foundational IPC mechanisms with advanced concurrency tools, essential for scalable and distributed applications.
