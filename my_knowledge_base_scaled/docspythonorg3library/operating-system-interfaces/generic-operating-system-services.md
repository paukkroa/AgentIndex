# Generic Operating System Services¶

This chapter documents Python’s **cross-platform OS service modules**, providing standardized interfaces for core system operations like file/directory management, process control, timing, and logging. The **`os` module** offers Unix/C-inspired functions for file descriptors, process management, system info, and randomness, including terminal size queries and Linux-specific extended attributes. The **`io` module** delivers high-level stream handling (text/binary/raw I/O) with buffering, encoding support, and performance optimizations for multi-threading. The **`time` module** standardizes time access (clocks, timezones) and conversions, while **`logging`** and **`logging.config`** enable structured log management with handlers, formatters, and thread-safe configurations. These modules abstract OS-specific complexities, ensuring portability across Unix, Windows, and other platforms.
