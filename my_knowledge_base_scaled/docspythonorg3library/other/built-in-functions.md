# Built-in Functions¶

The **Built-in Functions** page in Python’s standard library documents core functions preloaded in the interpreter, organized alphabetically for quick reference. These functions enable fundamental operations like arithmetic (`abs()`, `pow()`), type conversion (`int()`, `str()`), iteration (`map()`, `zip()`), and object inspection (`dir()`, `type()`). Key features include:
- **Mathematical & Type Functions**: `abs()` computes absolute values, `complex()` creates complex numbers, and `bool()` converts values to boolean.
- **Iteration & Mapping**: `map()`, `filter()`, and `zip()` facilitate functional programming; `enumerate()` adds counters to iterables.
- **Object Introspection**: `dir()`, `callable()`, and `hasattr()` inspect objects dynamically; `vars()` returns `__dict__` for inspecting attributes.
- **Asynchronous Support**: `aiter()` and `anext()` enable async iteration (Python 3.10+).
- **Safety & Control**: `eval()`/ `exec()` execute dynamic code (with caution); `input()` handles user input; `breakpoint()` integrates with debuggers.
